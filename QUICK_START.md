---

## 🆘 If You Get Stuck

1. **Don't understand concept?**
   - Re-read the README completely
   - It's written to be comprehensive, not brief

2. **Can't solve problem?**
   - Read the explanation (not code)
   - Understand the approach
   - Then code it yourself

3. **Code doesn't compile?**
   - Check BEST_PRACTICES.md for syntax rules
   - Make sure all variables initialized
   - Check for typos

4. **Tests don't pass?**
   - Trace your code manually
   - Check edge cases: 0, negative, large
   - Compare logic with solution
